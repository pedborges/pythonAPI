## <h1>pythonAPI</h1>

Consiste em verificar as variáveis de ambiente "frase1", "frase2" e "frase3" e devolve-las em tela a partir do route "/".
## ℹ️ Importante:
Lembre-se de alterar/criar o seu user e pwd secrets quando der o fork da aplicação.
